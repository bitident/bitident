How to authenticate with a Metaverse Avatar using bitident

Use your Metaverse digital identity for the decentralized authentication mechanism of bitident. The following steps will guide you through your first Blockchain powered authentication.

  1. Open the website or application you want to authenticate to

    You should be presented with a QR-Code or a text that contains the request parameters

  2. Open your bitident compatible wallet and use the Authentication tool to scan the request

    You should now see the request parameters. Carefully check the request origin and URL

  3. Sign and send the request

    Your wallet will automatically send it to the origin URL.

  4. Check your authentication status

    The request origin should be able to verify your signed request and accept your authentication
